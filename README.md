# team_c

본 프로젝트는 태양광 발전 시스템의 온도 상승으로 인한 에너지 효율 감소를 해결하기 위해 냉각장치를 활용하여 태양광 발전 효율을 증가시키는 장치이다.

장치는 '냉각장치를 이용한 태양광 효율 증가장치'로 펠티어 소자를 이용하여 태양광 패널의 온도를 저하시켜준다.

구체적인 흐름도는 아래 이미지와 같다.
